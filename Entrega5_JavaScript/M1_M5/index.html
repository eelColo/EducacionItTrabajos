Ejercicio 1: Calculadora de Punto de equilibrio

El punto de equilibrio es un indicador basico de la rentabilidad de un negocio. Permite medir las ganancias y pérdidas de un proceso productivo.
El punto de equilibrio es la cantidad de ventas que un negocio debe tener para no perder dinero.
